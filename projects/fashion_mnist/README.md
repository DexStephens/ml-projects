# FashionMNIST Classifier

PyTorch walkthrough for datasets, transforms, dataloaders, a simple neural network, and inference.

## Run

From the repo root:

```bash
uv sync
source .venv/bin/activate
python projects/fashion_mnist/main.py
```

FashionMNIST downloads to `projects/fashion_mnist/data/` (gitignored).

## Files

- `main.py` — primary tutorial script
- `custom_image_dataset.py` — starter custom `Dataset` implementation
